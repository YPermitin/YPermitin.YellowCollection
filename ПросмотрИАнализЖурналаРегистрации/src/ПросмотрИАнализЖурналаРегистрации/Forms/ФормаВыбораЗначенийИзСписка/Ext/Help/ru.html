Выбор значений для отбора

В данном окне можно указать те значения, которые будут участвовать в отборе. В списке представлены все возможные значения отбора.

Участвовать в отборе будут те значения, которые отмечены флажком. Установка флажка для всех значений означает отключение отбора по редактируемому значению.
